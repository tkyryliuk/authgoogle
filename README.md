# Authgoogle
GOOGLE OpenID connect auth module

This is a [SimpleSAMLphp](https://github.com/simplesamlphp/simplesamlphp) module based on the one created by
[sylvainmed](https://github.com/sylvainmed/simplesamlphp/tree/master/modules/authgoogle)

It was minimally changed to keep compactibility with SimpleSAMLphp version 1.14
